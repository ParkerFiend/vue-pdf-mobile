# vue-pdf-mobile

基于 vue-pdf 的二次封装

集成了基础的双指缩放功能

## props 入参

|   字段    |   字段名称   |   类型   | 是否必传 | 默认值 |   描述   |
| :-------: | :----------: | :------: | :------: | :----: | :------: |
|    url    |   pdf 链接   | `string` |    是    |   无   |          |
| scaleRate | 双指缩放速度 | `number` |    否    |  100   | 越大越慢 |
